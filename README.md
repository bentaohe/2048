# Phaser3 开发 2048 游戏
运行环境 
- node 10.4（可用 nvm 切换 node 版本）
- webpack 4.12
- phaser 3.10.1

整体开发中...

## 功能
- 支持 WSAD 及方向键操作
- 支持 移动端滑动操作

## 运行本项目
```bash
# 克隆项目
git clone https://github.com/liurongqing/2048.git

# 安装依赖
npm install # or yarn 

# 运行项目
npm start # or yarn start

# 打包生产
npm run prod # or yarn run prod
```
